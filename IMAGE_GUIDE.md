# Wardrobe Stylist - 图片采集规范

本文档指导您拍摄和上传适合当前 try-on 系统的衣物图片。

## 1. 拍摄环境要求

### 背景
- **推荐**：纯色背景（白色、浅灰、浅色）
- **避免**：杂乱背景、复杂图案、深色背景
- **原因**：系统使用固定模板叠加，杂乱背景会干扰视觉效果

### 光线
- **推荐**：均匀自然光或柔光灯
- **避免**：强烈阴影、过曝、色偏
- **原因**：光线不均会导致衣物颜色失真

## 2. 不同类别拍摄建议

### Top (上衣)
- **角度**：正面垂直拍摄，镜头与衣物平行
- **构图**：衣物平铺或悬挂，肩部对齐画面顶部
- **留白**：衣物上方留少量空间，下方留较多空间（预留腰部位置）
- **示例**：
  ```
  [肩部位置]
  ┌─────────────┐
  │   上衣      │
  │             │
  └─────────────┘
  [预留腰部空间]
  ```

### Bottom (裤子/裙子)
- **角度**：正面垂直拍摄
- **构图**：腰部对齐画面顶部，裤脚向下延伸
- **留白**：上方预留腰部位置，下方留少量空间
- **示例**：
  ```
  [预留腰部位置]
  ┌─────────────┐
  │   裤子      │
  │             │
  └─────────────┘
  [裤脚位置]
  ```

### Shoes (鞋子)
- **角度**：正面或微俯视角度
- **构图**：鞋子底部对齐画面底部
- **留白**：上方留较多空间，下方贴近底部
- **示例**：
  ```
  [上方空间]
  ┌─────────────┐
  │             │
  │   鞋子      │
  └─────────────┘
  [底部对齐]
  ```

### Outerwear (外套)
- **角度**：正面垂直拍摄
- **构图**：类似上衣，但可稍宽松
- **留白**：肩部对齐顶部，下方预留空间
- **注意**：外套图片应比上衣稍大，便于覆盖

## 3. 图片规格建议

### 尺寸
- **推荐**：800x1200 像素（2:3 比例）
- **最小**：400x600 像素
- **最大**：2000x3000 像素

### 格式
- **推荐**：JPEG 或 PNG
- **避免**：GIF（动画）、WebP（兼容性）

### 文件大小
- **推荐**：100KB - 500KB
- **最大**：5MB（系统限制）

## 4. 不适合当前系统的图片

### ❌ 不适合
1. **多角度拍摄**：当前只支持正面视图
2. **模特穿着**：当前只支持衣物平铺/悬挂图
3. **复杂背景**：会干扰 try-on 效果
4. **阴影严重**：影响衣物颜色识别
5. **过度裁剪**：无法识别衣物完整轮廓
6. **低分辨率**：模糊影响视觉效果

### ✅ 适合
1. **正面平铺**：衣物完全展开
2. **纯色背景**：白色、浅灰最佳
3. **光线均匀**：无强烈阴影
4. **完整轮廓**：衣物边缘清晰可见
5. **适中尺寸**：800x1200 像素左右

## 5. 手动调整指南

如果 try-on 预览效果不理想，可以使用手动调整功能：

### Offset X (水平偏移)
- **范围**：-50px 到 +50px
- **用途**：左右微调衣物位置
- **建议**：先调整这个，让衣物居中

### Offset Y (垂直偏移)
- **范围**：-50px 到 +50px
- **用途**：上下微调衣物位置
- **建议**：调整衣物与模特的对齐

### Scale (缩放)
- **范围**：0.5x 到 1.5x
- **用途**：调整衣物大小
- **建议**：最后调整，确保衣物比例合适

## 6. 常见问题

### Q: 衣物图片有白边怎么办？
A: 系统会自动裁剪，但建议拍摄时尽量减少白边

### Q: 衣物位置不对怎么办？
A: 使用手动调整功能，微调 Offset X/Y

### Q: 衣物太大/太小怎么办？
A: 使用手动调整功能，调整 Scale

### Q: 什么图片最适合？
A: 纯色背景、正面平铺、光线均匀、完整轮廓的衣物图片
