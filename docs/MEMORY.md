# Wardrobe Stylist Memory

## Project summary
wardrobe-stylist 是一个用于整理衣柜中所有衣物、创建穿搭组合、并查看搭配效果的项目。

## Core product goal
帮助用户：
- 记录自己拥有的衣服
- 用结构化方式管理衣物
- 根据场景和风格自由搭配
- 最终看到搭配效果
- 后续可以加入 AI 辅助建议或试穿能力

## MVP roadmap
### Phase 1
衣物管理：
- add clothing item
- edit clothing item
- list clothing items
- filter by category / season / color / style

### Phase 2
穿搭组合：
- create outfit
- pick multiple clothing items into one outfit
- save and browse outfits

### Phase 3
预览体验：
- show outfit as combined visual result
- start with simple image-based preview
- AI try-on is optional and later

## Product principles
- Prefer practical utility over novelty
- Start with structured data and usable workflows
- Delay advanced AI until the core workflow works
- Build in phases, not all at once

## Likely domain entities
- clothing item
- outfit
- image
- tag
- category
- season
- style profile
- occasion

## Delivery expectation
一个阶段任务完成时，至少要能说明：
- 当前实现属于哪个 phase
- 用户现在可以完成什么动作
- 还缺哪些关键能力
- 如何验收当前版本
