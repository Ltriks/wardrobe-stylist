# 衣物图片采集规范与拍摄指引 (Phase 5.0)

## 1. 衣物图片怎么拍

### 1.1 背景要求
- **推荐背景**：纯白色或浅灰色背景（便于自动裁剪留白）
- **避免**：杂乱背景、图案背景、深色背景
- **技巧**：在白墙前拍摄，或使用白色床单/纸张铺垫

### 1.2 光线要求
- **推荐光线**：自然光（窗边）或均匀的室内灯光
- **避免**：强烈直射光（产生阴影）、昏暗环境（噪点多）
- **技巧**：避免闪光灯，保持光线均匀覆盖衣物

### 1.3 主体大小
- **推荐比例**：衣物占据画面 60%-80%
- **避免**：衣物太小（留白过多）、衣物太大（超出画面）
- **技巧**：拍摄时预留裁剪空间，系统会自动裁剪留白

### 1.4 角度要求
- **推荐角度**：正面平拍（90度垂直拍摄）
- **避免**：倾斜角度、俯拍、仰拍
- **技巧**：手机/相机保持水平，衣物平铺或悬挂平整

### 1.5 不同类型衣物拍摄建议

#### Top (上装：T恤、衬衫、毛衣等)
- **拍摄方式**：平铺或悬挂
- **重点**：展示领口、袖口、整体版型
- **避免**：褶皱过多、领口变形

#### Bottom (下装：裤子、裙子等)
- **拍摄方式**：平铺（裤子可对折展示裤腿）
- **重点**：展示裤腰、裤腿、长度
- **避免**：裤腿扭曲、裤腰不平整

#### Outerwear (外套：夹克、大衣等)
- **拍摄方式**：悬挂或平铺
- **重点**：展示领口、袖子、整体轮廓
- **避免**：袖子遮挡主体、领口变形

#### Shoes (鞋子)
- **拍摄方式**：平放，侧面或正面展示
- **重点**：展示鞋型、鞋底、颜色
- **避免**：鞋底朝上、角度倾斜

## 2. 哪些图片不适合当前系统

### 2.1 不适合的图片类型
- **多件衣物同框**：系统无法分离多件衣物
- **人物穿着衣物**：系统无法去除人物背景
- **模糊/失焦图片**：AI 分类和标准化会失败
- **极端角度**：系统难以识别衣物主体
- **纯色/纯白图片**：系统无法检测衣物边界
- **低分辨率图片**（< 500x500）：影响预览和 try-on 效果

### 2.2 建议重拍的情况
- 图片中有杂物、人物、背景杂乱
- 衣物褶皱严重，影响识别
- 光线不均匀，产生强烈阴影
- 衣物颜色与背景过于接近

## 3. 当前批量上传推荐的命名和拍摄方式

### 3.1 文件命名建议
- **推荐格式**：`类别_颜色_描述.jpg`
- **示例**：
  - `top_white_tshirt.jpg`
  - `bottom_blue_jeans.jpg`
  - `outerwear_black_jacket.jpg`
  - `shoes_white_sneakers.jpg`
- **好处**：系统会根据文件名自动建议类别和颜色

### 3.2 拍摄流程建议
1. **准备阶段**：
   - 铺好白色背景
   - 调整光线均匀
   - 准备待拍摄衣物

2. **拍摄阶段**：
   - 每件衣物单独拍摄
   - 保持手机/相机水平
   - 确保衣物占据画面 60%-80%

3. **上传阶段**：
   - 批量选择所有图片
   - 系统会自动分类、标准化、检测重复
   - 在待确认页面检查结果

### 3.3 批量拍摄技巧
- **统一背景**：所有图片使用相同背景，便于标准化
- **统一光线**：同一时间段拍摄，光线一致
- **统一角度**：保持相同拍摄距离和角度
- **命名规范**：按类别分组拍摄，便于文件命名

## 4. 未来 personal template / avatar 拍摄建议

### 4.1 Personal Template (个人模板)
- **用途**：用于虚拟试衣、搭配预览
- **拍摄要求**：
  - 纯色背景（推荐白色或浅灰）
  - 正面、侧面、背面三视图
  - 身体比例准确（避免广角畸变）
  - 穿着贴身衣物（便于虚拟试衣）

### 4.2 Avatar (虚拟形象)
- **用途**：用于个性化展示、社交分享
- **拍摄要求**：
  - 正面半身照或全身照
  - 表情自然、姿势放松
  - 光线均匀、背景简洁
  - 分辨率足够高（建议 1024x1024 以上）

### 4.3 拍摄预留
- **front / back / side 视图**：建议提前规划三视图拍摄
- **标准化尺寸**：建议统一为正方形（便于后续处理）
- **文件命名**：建议按 `template_front.jpg`、`template_back.jpg` 命名

## 5. 当前系统能力与限制

### 5.1 系统能做什么
- ✅ 自动裁剪留白
- ✅ 自动居中衣物主体
- ✅ 统一输出 512x512 正方形图片
- ✅ 基于类别 + 颜色的重复检测
- ✅ 基于 MobileNet 的粗分类（top/bottom/outerwear/shoes/accessory）

### 5.2 系统不能做什么
- ❌ 识别多件衣物同框的图片
- ❌ 去除人物背景
- ❌ 修复模糊/失焦图片
- ❌ 自动合并重复衣物
- ❌ 虚拟试衣（未来功能）

## 6. 快速检查清单

上传前检查：
- [ ] 图片中只有单件衣物
- [ ] 背景简洁（白色/浅灰最佳）
- [ ] 光线均匀，无强烈阴影
- [ ] 衣物占据画面 60%-80%
- [ ] 图片清晰，无模糊
- [ ] 文件名包含类别和颜色（可选）

上传后检查：
- [ ] 系统正确识别类别和颜色
- [ ] 标准化图片裁剪了留白
- [ ] 无重复提示（或确认重复合理）
- [ ] 图片预览清晰可见
